COFFEE BREAK

This site details all nine locations on campus at which one can acquire coffee. Each page contains the hours of operation, a short description, and a jQuery photo slideshow. An interactive map is included to show where each cafe resides on campus. Both horizontal and vertical scrolling were employed to complete the effect of the site being iPad compatible.

Built in HTML5, CSS3, jQuery
