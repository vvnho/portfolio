SPIRAL COUNTER

Through the use of Processing, an open source programming language for a digital sketchbook of sorts, I created a primitive form of data visualization where my code would analyze content depending on the input type.

Upon running, the code will open two seperate windows: a control panel and a drawing window. From the control dock, the user can specify parameters that will chage the output onto the drawing window. The resulting arcs revolve around the center of the drawing window.
