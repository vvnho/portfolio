ALCOHOL IMPAIRMENT CHART REDESIGN

BACKGROUND

The state of California encloses a copy of the Alcohol Impairment Chart when mailing car registration renewals, new driver's licenses, etc. Unfortunately, the chart is cluttered and intimidates the user, resulting in fewer people wanting to read through and digest the information.

Constraints included keeping the print to greyscale (with printing costs in mind), maintaining the original size to fit easily in a #10 standard envelope, and only using one side of the paper. Upon reducing the copy to what was most relevant, I explored with design principles of visual hierarchy and color blocking to create drafts that were easier for the user to navigate.

Through user testing and several refinements, I found that a vertical arrangement was more successful than a horizontal format. The vertical format opened up more space and allowed for easier reading through visual hierarchy. Using icons to depict information also helps to reduce large blocks of copy for the user to undestand.
